# PetCare

Sistema desenvolvido em equipe no curso de Desenvolvimento de Sistemas do SENAI. O PetCare organiza o cadastro de clientes, animais de estimação e serviços de um pet shop, facilitando a gestão de atendimentos.

## Funcionalidades
- Cadastro de clientes
- Cadastro de pets
- Cadastro de serviços
- Agendamento de atendimentos
- Interface simples e fácil de usar

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Bootstrap
- Modelagem de dados
- Metodologias ágeis (Scrum e Sprints)

## Como rodar
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Navegue pelas páginas e funcionalidades
